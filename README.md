# Vasant Shobha Foundation Website

The Vasant Shobha Foundation, established in India, aims to improve educational access for children and adults in underserved communities. Since its inception in March 2024, the non-profit has expanded to multiple villages, hosting events and classes that empower local communities.

## Website Overview

The website serves as a platform to provide information about the foundation, its activities, and ways for users to get involved. It includes sections on available classes, team information, and an interactive map for events.

## Main Features

### 1. Classes
- **Description**: Detailed descriptions of classes available for youth and adults.
- **Search and Sort**: Mechanisms to search and sort classes based on various criteria.
- **Sign-Up**: Option for users to sign up for classes.

### 2. Get Involved
- **Interactive Map**: Allows users to find events near them.
- **Contact Box**: Enables users to get in touch with the foundation for volunteering or donations.

### 3. About
- **Foundation Information**: Details about the foundation’s mission, vision, and history.

### 4. Our Team
- **Team Profiles**: Information about the foundation's team members.

### Additional Features
- **Navigation**: Intuitive navigation menu to access various sections of the website.
- **Responsive Design**: Ensures the website is accessible on all devices.


### Folder Descriptions

- **/assets**: Contains static assets like images, stylesheets, and scripts.
- **/supporting**: Reusable UI components such as the header, footer, and navigation bar.
- **/pages**: Each file represents a different page on the website.
- **/utils**: Utility functions and API call handlers.
- **/data**: Static JSON files for initial data like classes and events.

## Navigation

The website features a top navigation bar that allows users to easily access the main sections:
- Home
- About
- Classes
- Get Involved
- Our Team

## Functionalities

1. **Dynamic Class Listings**: Classes are dynamically loaded and can be filtered and sorted.
2. **Interactive Map**: Displays events based on the user's location.
3. **Contact Form**: Allows users to send messages directly through the website.
4. **Responsive Design**: Adjusts layout and functionality for mobile, tablet, and desktop views.

## Setup Instructions

To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/VasantShobhaWebsite.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd VasantShobhaWebsite
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Run the development server**:
   ```bash
   npm start
   ```

Visit the website at [vasantshobha.org](https://vasantshobha.org)
